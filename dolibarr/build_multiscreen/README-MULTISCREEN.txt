# TakePOS Multi-Screen Customer Display

Added in this package:
- /takepos/customer_display.php
- Customer Screen floating button on cashier page
- Live secondary-screen sync using BroadcastChannel + localStorage

Usage:
1. Open TakePOS on the cashier screen.
2. Click **Customer Screen**.
3. Move the new window to the second monitor and make it fullscreen.
4. The customer screen updates automatically as the cashier cart changes.

Best suited for: same PC, same browser, dual-monitor setup.
