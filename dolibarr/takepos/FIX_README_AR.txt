TakePOS Full Fix Package
========================

التركيب:
1) خذ نسخة احتياطية من ملفات Dolibarr وقاعدة البيانات.
2) ارفع مجلد takepos الموجود داخل هذا البكج فوق مجلد htdocs/takepos الحالي.
3) نفّذ ملف SQL المرفق يدوياً: sql/takepos_full_fix_manual_db_upgrade.sql
4) إذا كانت بادئة الجداول لديك ليست llx_ عدّلها داخل ملف SQL قبل التنفيذ.
5) إذا كان رقم entity ليس 1 عدّل السطر: SET @takepos_entity = 1;
6) عدّل سعر الدولار في السطر: SET @takepos_usd_rate = '1.410000'; حسب السعر المطلوب.
7) امسح كاش المتصفح وكاش Dolibarr إن وجد ثم جرّب TakePOS.

التعديلات الرئيسية:
- إصلاح البحث بالماسح/الباركود وإضافة المنتج فوراً للفاتورة.
- دعم البحث برقم المنتج/ref والباركود الأساسي والباركودات الإضافية.
- إظهار زر فتح الدرج وزر الدفع Visa/Card من الشاشة الرئيسية.
- تفعيل شاشة نظرة عامة على المخزون وإظهار منتجات batch/lot بدلاً من إخفائها.
- إلغاء الفواتير المعلقة عند إغلاق الشفت وربط الفواتير بالشفت.
- إضافة جداول ربط الفاتورة بالشفت ومعلومات الدفع بعملة أجنبية.
- دعم أكثر من باركود لنفس المنتج عبر شاشة الباركودات المتعددة.
- إضافة شاشة نسب الضرائب لإدخال نسب ضريبية جديدة.
- تحويل العملة الأساسية إلى JOD عبر SQL وإضافة USD كعملة دفع إضافية مع سعر صرف.
- إضافة تقارير: الشيكات، الذمم المدينة، الذمم الدائنة، حركة/سرعة الأصناف، تاريخ حركات المخزون، الأصناف قريبة الانتهاء، والمبيعات التفصيلية/الإجمالية.
- إصلاح أسماء جدول/حقول الشيكات في Dashboard Service.

تم فحص PHP syntax لجميع ملفات PHP داخل المجلد ولم تظهر أخطاء syntax.
