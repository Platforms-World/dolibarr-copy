ما فائدة هذه الشاشة؟

يعرض دفتر المصاريف جميع المصروفات المسجلة بطريقة تساعد على المراجعة والتدقيق والمتابعة.

كيف تستخدمها؟

  • استخدم الفلاتر حسب التاريخ أو الفئة أو الحالة.
  • راجع تفاصيل كل حركة مصروف.
  • افتح السجل المطلوب عند الحاجة للتعديل أو المراجعة.
  • تابع الإجماليات الناتجة عن البحث الحالي.
  • استخدم الطباعة أو التصدير إذا كانت متوفرة.

نصائح مهمة

  • خصص فترة زمنية واضحة للحصول على نتائج أدق.
  • استخدم دفتر المصاريف للمراجعة الدورية مع الإدارة المالية.
  • قارن الإجماليات مع النقد أو المستندات الورقية عند الحاجة.

English Guide

What is this screen for?

The expense ledger shows all recorded expenses in a format suitable for review, audit, and follow-up.

How do you use it?

  • Use filters by date, category, or status.
  • Review the details of each expense entry.
  • Open a record when you need to inspect or edit it.
  • Track totals based on the current filter.
  • Use print or export if available.

Helpful tips

  • Choose a clear date range for more accurate results.
  • Use the ledger for periodic financial review.
  • Compare totals with cash or paper documents when needed.
