ما فائدة هذه الشاشة؟

تضم هذه الشاشة الإعدادات العامة الأساسية التي تؤثر على سلوك النظام وطريقة تشغيله.

كيف تستخدمها؟

  • افتح الإعداد المطلوب بعناية.
  • راجع القيمة الحالية قبل تغييرها.
  • عدّل فقط ما تفهم تأثيره.
  • احفظ التغييرات ثم اختبر النتيجة.
  • وثّق الإعدادات المهمة داخليًا.

نصائح مهمة

  • يفضل أن يقوم بهذه التعديلات المسؤول أو المدير.
  • أي تغيير عام قد يؤثر على جميع المستخدمين.

English Guide

What is this screen for?

This screen contains core general settings that affect how the system behaves and operates.

How do you use it?

  • Open the required setting carefully.
  • Review the current value before changing it.
  • Only change settings you understand.
  • Save the changes and test the result.
  • Document important settings internally.

Helpful tips

  • These changes are best handled by an administrator or manager.
  • Any global change may affect all users.
