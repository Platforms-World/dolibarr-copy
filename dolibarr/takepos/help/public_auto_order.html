English Guide

What is this screen for?

This page allows the customer or user to place an order directly if self-ordering is enabled.

How do you use it?

  • Open the self-order page.
  • Choose items and quantities.
  • Review the cart or final order.
  • Confirm submission.
  • Track processing in the main system.

Helpful tips

  • Make sure items and prices are clearly shown.
  • Test the full flow from order to fulfillment.
