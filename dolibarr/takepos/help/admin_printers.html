الشرح بالعربية

ما فائدة هذه الشاشة؟

تستخدم هذه الشاشة لتعريف الطابعات وضبط إعداداتها بحيث يتم طباعة الإيصالات أو الأوامر بالشكل الصحيح.

كيف تستخدمها؟

  • اختر الطابعة أو أضف طابعة جديدة.
  • راجع نوع الطابعة وإعدادات الاتصال.
  • اختبر الطباعة.
  • احفظ الإعداد بعد التأكد من النتيجة.

نصائح مهمة

  • نفّذ طباعة تجريبية بعد أي تعديل.
  • تأكد من أن كل تيرمنال مرتبط بالطابعة الصحيحة.

English Guide

What is this screen for?

Use this screen to define printers and configure them so receipts or order slips print correctly.

How do you use it?

  • Choose the printer or add a new one.
  • Review printer type and connection settings.
  • Run a print test.
  • Save settings after confirming the result.

Helpful tips

  • Perform a test print after every change.
  • Make sure each terminal is linked to the correct printer.
