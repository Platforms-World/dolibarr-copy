ما فائدة هذه الشاشة؟

تستخدم هذه الشاشة لتسجيل المصاريف التشغيلية اليومية مثل النقل أو الضيافة أو المصاريف الإدارية.

كيف تستخدمها؟

  • اختر فئة المصروف.
  • أدخل المبلغ وطريقة الدفع والوصف.
  • اربط المصروف بالمورد أو المرجع إذا لزم الأمر.
  • احفظ العملية لتسجيلها في النظام.
  • راجع العمليات الأخيرة للتأكد من التسجيل الصحيح.

نصائح مهمة

  • اكتب وصفًا واضحًا ليسهل الرجوع للمصروف لاحقًا.
  • استخدم الفئة الصحيحة لتحسين التقارير.
  • راجع المبلغ قبل الحفظ خصوصًا عند الدفع النقدي.

English Guide

What is this screen for?

Use this screen to record daily operating expenses such as transport, hospitality, or administrative costs.

How do you use it?

  • Choose the expense category.
  • Enter amount, payment method, and description.
  • Link the expense to a supplier or reference when needed.
  • Save the transaction to register it in the system.
  • Review recent entries to confirm correct posting.

Helpful tips

  • Write a clear description for easier future review.
  • Use the correct category to improve reporting.
  • Double-check the amount before saving, especially for cash payments.
