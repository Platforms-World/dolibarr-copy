English Guide

What is this screen for?

This screen lets you define loyalty program rules such as how points are earned or redeemed.

How do you use it?

  • Review current rules.
  • Update ratios or point conditions when needed.
  • Save the changes.
  • Test a sale to confirm correct behavior.

Helpful tips

  • Any change here affects all customers.
  • Document the loyalty policy after each update.
