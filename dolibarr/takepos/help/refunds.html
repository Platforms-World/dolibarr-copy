ما فائدة هذه الشاشة؟

تستخدم هذه الشاشة لإرجاع أصناف تم بيعها سابقًا سواء بشكل كلي أو جزئي حسب السياسة المتبعة.

كيف تستخدمها؟

  • ابحث عن الفاتورة أو العملية الأصلية.
  • اختر الصنف أو الأصناف المراد إرجاعها.
  • حدد الكمية المسترجعة والسبب إذا كان مطلوبًا.
  • راجع المبلغ المستحق للعميل.
  • احفظ العملية ثم اطبع إيصال المرتجع.

نصائح مهمة

  • تأكد من اختيار الفاتورة الأصلية الصحيحة.
  • تحقق من الكمية التي سبق إرجاعها لتجنب التكرار.
  • وضح سبب الإرجاع لتحسين المتابعة.

English Guide

What is this screen for?

Use this screen to return items that were sold previously, whether full or partial, according to policy.

How do you use it?

  • Find the original invoice or sale.
  • Select the item or items to be returned.
  • Enter the returned quantity and reason if required.
  • Review the amount to be refunded.
  • Save the transaction and print the refund receipt.

Helpful tips

  • Make sure you selected the correct original invoice.
  • Check previously returned quantities to avoid duplication.
  • Record a clear return reason for better follow-up.
