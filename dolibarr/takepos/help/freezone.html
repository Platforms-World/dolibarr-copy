ما فائدة هذه الشاشة؟

تتيح هذه الشاشة إضافة بند حر أو عنصر نصي غير معرف مسبقًا في النظام عند الحاجة.

كيف تستخدمها؟

  • أدخل اسم البند أو وصفه.
  • حدد السعر والكمية.
  • راجع قيمة البند قبل إضافته.
  • أضفه إلى الفاتورة الحالية.
  • أكمل البيع كالمعتاد.

نصائح مهمة

  • استخدم هذه الميزة فقط عند الحاجة الفعلية.
  • اكتب وصفًا واضحًا ليسهل تتبع العملية لاحقًا.

English Guide

What is this screen for?

This screen allows you to add a free-text item or a non-predefined entry when needed.

How do you use it?

  • Enter the item name or description.
  • Set the price and quantity.
  • Review the value before adding it.
  • Add it to the current invoice.
  • Continue the sale as usual.

Helpful tips

  • Use this feature only when truly necessary.
  • Write a clear description so the transaction remains easy to track later.
