ما فائدة هذه الشاشة؟

تستخدم شاشة المستخدمين لإدارة الحسابات والصلاحيات وربط كل مستخدم بالدور المناسب له.

كيف تستخدمها؟

  • أضف مستخدمًا جديدًا أو افتح مستخدمًا موجودًا.
  • حدّث البيانات الأساسية عند الحاجة.
  • اضبط الصلاحيات حسب الدور الوظيفي.
  • احفظ ثم اختبر الصلاحيات عمليًا.
  • راجع المستخدمين دوريًا.

نصائح مهمة

  • امنح أقل صلاحية لازمة فقط.
  • أوقف الحسابات غير المستخدمة لحماية النظام.

English Guide

What is this screen for?

Use the users screen to manage accounts, permissions, and the role assigned to each user.

How do you use it?

  • Create a new user or open an existing one.
  • Update the basic information when needed.
  • Set permissions according to job role.
  • Save and test permissions in practice.
  • Review users periodically.

Helpful tips

  • Grant the minimum necessary access.
  • Disable unused accounts to protect the system.
