ما فائدة هذه الشاشة؟

تستخدم شاشة التيرمنال لإعداد جهاز نقطة البيع وربطه بالفرع والطابعة والخيارات التشغيلية المناسبة.

كيف تستخدمها؟

  • افتح التيرمنال المطلوب.
  • راجع اسم الجهاز وربطه بالمتجر.
  • حدد الخيارات التشغيلية المناسبة.
  • احفظ ثم اختبر البيع والطباعة.

نصائح مهمة

  • لكل جهاز إعداداته الخاصة، فانتبه عند التعديل.
  • اختبر الإعداد بعد الحفظ مباشرة.

English Guide

What is this screen for?

Use the terminal screen to configure the POS device and connect it to the store, printer, and operational options.

How do you use it?

  • Open the required terminal.
  • Review device name and store assignment.
  • Set the proper operational options.
  • Save and test sales and printing.

Helpful tips

  • Each terminal may have its own settings, so edit carefully.
  • Test immediately after saving.
