ما فائدة هذه الشاشة؟

تعرض لوحة المؤشرات ملخصًا بصريًا لأداء المبيعات والمخزون والحركة التشغيلية لمساعدة الإدارة على اتخاذ القرار.

كيف تستخدمها؟

  • راجع المؤشرات الرئيسية فور فتح الصفحة.
  • استخدم الفلاتر الزمنية عند توفرها.
  • افحص الرسوم البيانية والاتجاهات.
  • تابع المنتجات الأعلى مبيعًا أو التنبيهات المهمة.
  • انتقل للتقارير التفصيلية عند الحاجة.

نصائح مهمة

  • ابدأ يومك بمراجعة المؤشرات السريعة.
  • استخدمها لاكتشاف المشاكل قبل أن تتفاقم.

English Guide

What is this screen for?

The dashboard provides a visual summary of sales, stock, and operational activity to support decision-making.

How do you use it?

  • Review key indicators as soon as you open the page.
  • Use time filters when available.
  • Inspect charts and trends.
  • Track top-selling items and important alerts.
  • Open detailed reports when needed.

Helpful tips

  • Start the day by checking the main indicators.
  • Use it to spot issues before they grow.
