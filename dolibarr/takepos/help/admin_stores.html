الشرح بالعربية

ما فائدة هذه الشاشة؟

تستخدم هذه الشاشة لتعريف الفروع أو المتاجر التي يعمل عليها النظام وتنظيم البيانات بحسب كل موقع.

كيف تستخدمها؟

  • أضف متجرًا أو افتح متجرًا موجودًا.
  • أدخل البيانات الأساسية الخاصة بالموقع.
  • اربط المستخدمين أو الأجهزة حسب الحاجة.
  • احفظ الإعدادات ثم اختبرها.

نصائح مهمة

  • استخدم أسماء واضحة للفروع.
  • راجع الربط بين الفروع والمستخدمين بانتظام.

English Guide

What is this screen for?

Use this screen to define stores or branches and organize system data by location.

How do you use it?

  • Create a new store or open an existing one.
  • Enter the basic location details.
  • Link users or terminals when required.
  • Save the settings and test them.

Helpful tips

  • Use clear branch names.
  • Review store-user assignments regularly.
