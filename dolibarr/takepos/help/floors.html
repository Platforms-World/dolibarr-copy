ما فائدة هذه الشاشة؟

تستخدم هذه الشاشة عادة في المطاعم أو المقاهي لتنظيم الطاولات والأدوار وإدارة الطلبات حسب الموقع.

كيف تستخدمها؟

  • اختر الدور أو القسم المناسب.
  • افتح الطاولة المطلوبة.
  • أضف الطلبات الخاصة بهذه الطاولة.
  • انتقل بين الطاولات حسب حالة كل طلب.
  • راجع الطاولات المشغولة والفارغة بسرعة.

نصائح مهمة

  • سمِّ الطاولات بشكل واضح لتجنب الأخطاء.
  • استخدم الألوان أو الحالات إن كانت متاحة لتسريع العمل.

English Guide

What is this screen for?

This screen is commonly used in restaurants or cafés to organize floors, tables, and orders by location.

How do you use it?

  • Select the correct floor or section.
  • Open the target table.
  • Add the order items for that table.
  • Switch between tables based on order status.
  • Review occupied and free tables quickly.

Helpful tips

  • Name tables clearly to avoid mistakes.
  • Use available colors or statuses to speed up work.
