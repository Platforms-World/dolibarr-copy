ما فائدة هذه الشاشة؟

تساعدك هذه الشاشة على إدارة الأجهزة المرتبطة بالنظام مثل أدوات العرض أو التكاملات الأخرى.

كيف تستخدمها؟

  • أضف جهازًا جديدًا أو افتح جهازًا موجودًا.
  • راجع بيانات الاتصال أو الربط.
  • اختبر الجهاز عند توفر خيار الاختبار.
  • احفظ التعديلات بعد التأكد.

نصائح مهمة

  • سمِّ الأجهزة بأسماء واضحة.
  • اختبر الأجهزة بعد أي تغيير تقني.

English Guide

What is this screen for?

This screen helps manage devices connected to the system such as displays or other integrations.

How do you use it?

  • Add a new device or open an existing one.
  • Review connection or binding details.
  • Run a test if the option is available.
  • Save changes after confirmation.

Helpful tips

  • Use clear names for devices.
  • Test devices after any technical change.
